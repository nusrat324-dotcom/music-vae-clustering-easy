# Music VAE Clustering

This repository contains the complete Neural Networks course project:
“VAE for Hybrid Language Music Clustering”
The project includes:
Easy Task: Basic VAE + K-Means clustering
Medium Task: Convolutional VAE + Audio/Lyrics hybrid features + multiple clustering methods
Hard Task: Beta-CVAE + Multi-modal clustering using audio, lyrics, and genre information
## Structure
- src/ : model and clustering code
- results/ : outputs, plots, metrics
